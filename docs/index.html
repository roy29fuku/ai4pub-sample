> OVERVIEW

概要

$ cat symposium_overview.txt

生成AIやAIエージェントの急速な発展により、学術出版のあり方が大きく変わろうとしています。

著者がAI、編集部もAIという未来において、研究者と出版社はどのような役割を担うべきでしょうか？

本シンポジウムでは、AI時代の学術出版の課題と可能性について、第一線の研究者とともに議論します。
