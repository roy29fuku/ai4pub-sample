OVERVIEW

生成AI、そしてAGI（汎用人工知能）の時代が到来しつつある今、学術出版の世界も大きな変革期を迎えています。

AIが論文を執筆し、AIが査読を行い、AIが編集作業を担う。そんな未来は、もはやSFの世界の話ではありません。 本シンポジウムでは、AI研究の最前線で活躍する研究者、学術出版に携わる編集者、そしてAI Agentsの開発者が一堂に会し、 学術出版の未来について熱い議論を交わします。
