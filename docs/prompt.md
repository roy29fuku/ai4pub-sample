# 学会・シンポジウムのLPを作成してください

## 実装方針

### 技術要件
- 単一HTMLファイル（ペライチLP）
- TailwindCSS（CDN経由）を使用
- レスポンシブ対応（PC/タブレット/スマホ）
- 外部ライブラリは最小限に

### デザイン方針
- モダンでプロフェッショナルな印象
- セクションごとに視覚的な区切り
- スピーカーカードは写真・所属・役職・SNSリンクを含む
- プログラムはタイムライン形式で見やすく
- アクセス情報にはGoogle Maps埋め込み

### 必須セクション
1. ヒーロー（タイトル、日時、場所）
2. 概要（OVERVIEW）
3. プログラム（PROGRAM）
4. 登壇者（SPEAKERS）
5. 参加費（FEE）
6. アクセス（ACCESS）
7. 主催・共催・後援（ORGANIZERS）
8. 参加登録ボタン（フッター固定 or セクション内）
